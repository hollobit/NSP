# AI 표준화 통합 대시보드

「제6차 국가표준기본계획(2026-2030)」과 「2026년도 국가표준시행계획」을
**AI 중심**으로 교차 분석하는 **단일 HTML 파일** 대시보드.

**라이브**: https://hollobit.github.io/NSP/  (GitHub Pages 활성화 후)

## 한눈에 보기

- **92개 세부 사업** × **18개 부처청** × **13개 중점추진과제** × **18개 중점 표준화 분야** 통합
- 2026년 총 **2,688억원** 예산을 AI 관점에서 해상도 3단계로 분석:
  - CT-1-2 AI 포커스 과제 전체 **195억** (정책 편성)
  - T01 AI 기술 태깅 사업 **940억** (기술 라벨)
  - **순AI 표준화 예산 157억** (line-item 수준 엄격 추출)
- **4계층 AI 분류** (ISO/IEC JTC1/SC42 WG 체계 참고):
  - AI 기반 표준 (Foundation) · AI 기술 표준 (Technology) · AI 응용 표준 (Application) · AI 활용 표준 (Utilization)
- 표준화 기구 46개 자동 발견 + JTC1~JTC5 참조 구조
- 단일 HTML 파일 (~500KB), CDN 의존, `file://` 오프라인 동작

## 분류 체계 (핵심 참조)

### 18개 중점 표준화 분야 (기본계획 p.12)
인공지능(T01) · 양자기술(T02) · 첨단제조(T03) · 차세대통신(T04) · 디스플레이(T05) ·
반도체(T06) · 미래차(T07) · 미래선박(T08) · 로봇(T09) · 첨단바이오(T10) ·
청정에너지(T11) · 핵심소재(T12) · 원자력(T13) · 이차전지(T14) · 메타버스(T15) ·
항공우주(T16) · 차세대보안(T17) · 시티버스/IoT(T18)

### 13개 중점 추진과제 (4대 분야)
- **D1 미래 핵심산업 및 AI 기반·융합 표준 선도** — CT-1-1 ~ CT-1-4 (4개)
- **D2 국민 체감 표준 인프라 확대** — CT-2-1 ~ CT-2-3 (3개)
- **D3 기술규제 대응 및 인증·인정 혁신** — CT-3-1 ~ CT-3-3 (3개)
- **D4 혁신적 표준 생태계 조성** — CT-4-1 ~ CT-4-3 (3개)

> ℹ️ 5차(2021-2025)는 12개 세부과제였고 6차에서 13개로 확대 개편되었으나, 문서 표제는 관행적으로 "12대 중점추진과제" 표기.

### AI 4계층 분류 체계
| 계층 | 정의 | 키워드 예 |
|---|---|---|
| **Foundation** (기반) | AI가 작동하는 인프라·데이터·모델 계층 | AI 인프라, 데이터셋, 파운데이션 모델 |
| **Technology** (기술) | AI 자체의 규범 — 신뢰성·안전성·평가 | AI 신뢰, 적합성평가, 윤리 |
| **Application** (응용) | 특정 산업·도메인에 AI 통합 | 의료 AI, 국방 AI, 피지컬 AI |
| **Utilization** (활용) | AI를 도구로 활용 | AX 전환, AI 기반 시스템 |

## 기술 스택

- **데이터 추출**: Python 3.11+, pdfplumber
- **프런트엔드**: Alpine.js 3 · Tailwind CSS (Play CDN) · D3 · Fuse.js (CDN 버전 고정)
- **빌드**: Python 스크립트가 JSON을 단일 HTML 템플릿에 인라인 주입 → `dist/index.html`
- **호스팅**: GitHub Pages (정적 단일 파일)

## 재현 방법 (개발자용)

원본 PDF는 저작권 문제로 이 리포에 포함되지 않습니다. 아래 절차는 로컬 빌드 기준.

```bash
# 1. 의존성
pip install pdfplumber

# 2. 파이프라인 (8단계)
bash scripts/build.sh
# ├─ [1] PDF → raw pages JSON
# ├─ [2] 기본 taxonomy (13 과제 + 18 부처) 시드
# ├─ [3] 5차→6차 이력 인코딩
# ├─ [4] 92 사업 atomic 추출
# ├─ [5] AI·기술·전략 분류 (4계층 AI 분류 포함)
# ├─ [6] 부처 메타 (법적근거 · 담당자 · 고유성과지표)
# ├─ [7] AI 연계 가중치 매트릭스
# └─ [8] 단일 HTML 렌더 → dist/index.html

# 3. 확인
open dist/index.html
```

## 알려진 제약

- **법적근거 미추출 3 부처** (중기부·조달청·질병청) — 원문 서식 다양성에 따른 long-tail
- **예산 unknown 4 사업** — "해당 없음"/"-" 표기 또는 표 병합 실패 케이스
- **표준화기구 NER 공적 편향** — "사실상표준기구" 한글 통칭 문맥 인식 한계 (향후 LLM 보조 예정)
- **AI 4계층 분류의 Utilization 0원** — "AI 활용" 키워드가 별도 예산 line으로 편성되지 않음
